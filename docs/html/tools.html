tools — golden_maker

獨立小工具集。目前唯一成員 golden_maker:製作對位 Golden Pattern(框選 Mark 截圖), 輸出 Golden PNG + RecipeInfo.xml 的 <M_AlignRoi> 片段,供 Gap #1 對位 pipeline 使用。

tools/golden_maker · C++ / OpenCV · 來源:docs/tools_程式完整說明.md

用途

取代 legacy PrjCfAoi 的 MIL MpatFind Pattern Match(CamProc.cs:307-485)。演算法重寫(非移植), 新架構用 OpenCV TM_CCOEFF_NORMED。golden_maker 負責離線製作對位樣板。

在 Gap #1 對位 pipeline 的位置

【離線,調 Recipe 前】golden_maker
代表幀框選 Mark → mark.png(Golden)+ <M_AlignRoi> XML 片段
人工貼入 RecipeInfo.xml
RecipeInfo.xml 的 <M_AlignRoi>
PatternPath / ReferX/Y / SearchWidth/Height
【線上,每片一次】LOAD_RECIPE
Control:讀 PatternPath → base64 嵌入 → 送 IP
network-clean:Golden PNG 以 base64 over TCP(不依賴對方硬碟)
CF_GRAB_START → CHECK_ALIGN

記憶體 decode Golden → 13 角 × TM_CCOEFF_NORMED + 拋物線 sub-pixel → ShiftX/Y

驗證鏈ip/src/align_verify.cpp(Stage 1:14/14 PASS)+ scripts/verify_alignment.py(Stage 2:8/8 PASS)。對位 pipeline 於 GB10 實機驗通(L3)。

兩種模式

模式觸發操作
GUI無 --mark-rect滑鼠拖拽框選;鍵 S 存 PNG + 印 XML、R 重設、Q/ESC 離開。大圖自動縮放(上限 2000px),存檔以 inv_scale 還原原始座標
CLI fallback帶 --mark-rect無桌面/SSH 用:直接指定 Mark 矩形(整數像素),headless 輸出

命令列參數

參數說明預設
--image <path>來源影像(OpenCV IMREAD_GRAYSCALE)必填
--output <path>輸出 Golden PNG 檔名mark.png
--mark-rect x,y,w,hCLI fallback:直接指定 Mark 矩形—(給了就走 CLI)
--search-margin NSearchWidth/Height = Mark 尺寸 × N3
用法範例
# GUI 模式
./golden_maker --image frame.png --output mark.png --search-margin 3
# CLI fallback(headless)
./golden_maker --image frame.png --mark-rect 1170,770,60,60

stdout 輸出 — AlignRoi XML 片段

tools/golden_maker/golden_maker.cpp:86–99審查:ReferX/Y = Mark 中心(非左上角);SearchW/H = 尺寸 × margin
static void print_align_roi_xml(const std::string& output_path,
                                int orig_x, int orig_y, int orig_w, int orig_h,
                                int search_margin) {
    int refer_x = orig_x + orig_w / 2;     // Mark 中心 X
    int refer_y = orig_y + orig_h / 2;     // Mark 中心 Y
    int sw      = orig_w * search_margin;   // 搜尋寬 = Mark 寬 × margin
    int sh      = orig_h * search_margin;
    std::printf("<!-- 貼入 RecipeInfo.xml <M_AlignRoi> -->\n");
    std::printf("<PatternPath>%s</PatternPath>\n",   output_path.c_str());
    std::printf("<ReferX>%d</ReferX>\n",             refer_x);
    std::printf("<ReferY>%d</ReferY>\n",             refer_y);
    std::printf("<SearchWidth>%d</SearchWidth>\n",   sw);
    std::printf("<SearchHeight>%d</SearchHeight>\n", sh);
}
語意提醒ReferX/Y 是 Mark 中心(不是左上角);SearchWidth/Height = Mark 尺寸 × --search-margin(預設 3),給移動足夠搜尋空間。框選 < 4×4 px 會拒絕。
