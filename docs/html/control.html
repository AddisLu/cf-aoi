Control — 控制平面 + 操作 UI

CF-AOI 的中間協調器:接上位機 TCP 命令(8787),協調 Grab/IP(8100/8200 JSON), 提供跨平台操作 UI。C# / Avalonia,同一份程式碼跑 Mac / Windows / Linux。

control/ · C# 12 / .NET 8 / Avalonia / CommunityToolkit.Mvvm · 來源:docs/control_程式完整說明.md

系統概述

取代 legacy 單機(WinForms + MIL + GPU),改成分散式控制平面。MIL/相機/GPU 邏輯全移除, 改以 TCP 呼叫遠端 Grab/IP。影像處理用 SixLabors.ImageSharp(不用 OpenCV,跨平台)。

network-clean跨機不共用檔案系統:配方傳 XML 內容、結果走 JSON、缺陷小圖以 PNG bytes(base64)over TCP。 IpClient 整行累積 bytes 後一次 UTF-8 解碼(不可逐 byte→char,否則中文亂碼)。

② RecipeStore.Select — 單一資料來源同步

control/src/Services/RecipeStore.cs:45–60 ↗審查:PrimaryZone 指向同一實例 → 跨 View 同步
public void Select(string name) {
    if (_inSelect) return;
    _inSelect = true;
    try {
        if (SelectedRecipe != name) SelectedRecipe = name;
        var ensure = _recipes.EnsureRecipeExists(name);
        Recipe = ensure.Recipe;
        if (Recipe.DetectRoiList.Count == 0) Recipe.DetectRoiList.Add(new ZoneSettingModel());
        PrimaryZone = Recipe.DetectRoiList[0];      // 共用實例
        RecipeSaving = _recipes.LoadRecipeSetting(name);
        RecipeReloaded?.Invoke();                  // 廣播 → 各 View 重綁
    } finally { _inSelect = false; }
}

③ ConnectionManager — 心跳 + 自動重連

control/src/Controllers/ConnectionManager.cs:54–83 ↗審查:IsBusy 跳過 / 失敗紅燈 + Disconnect 下回合重連
if (client.IsBusy) {
    ok = true;                            // 命令進行中 → 視為存活
} else {
    try {
        if (!client.IsConnected) {
            using var cc = new CancellationTokenSource(TimeSpan.FromSeconds(2));
            await client.ConnectAsync(node.Host, node.Port, cc.Token);
        }
        using var hb = new CancellationTokenSource(TimeSpan.FromSeconds(2));
        var resp = await client.CheckHealthAsync(hb.Token);
        ok = resp?["status"]?.GetValue<string>() == "OK";
    } catch { ok = false; }
    if (!ok) client.Disconnect();           // 關壞 socket → 下回合重連
}
